## Super User (Emergency Admin Password Reset)

The app includes a super user feature for emergency admin password resets when the admin account is locked out.

**Access**: Navigate to `/superuser/login` or click "Super User Access" on the login page

See [SUPER_USER_GUIDE.md](./SUPER_USER_GUIDE.md) for setup and usage instructions.

## Environment Variables

Create a `.env` file in the root directory with your Firebase credentials:

```
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_SUPER_USER_EMAIL=your-super-user@gmail.com
```

### Super User Configuration

The `VITE_SUPER_USER_EMAIL` variable specifies the Gmail address authorized to access the super user dashboard for emergency admin password resets. Only this email address will be allowed to sign in as a super user.

**Important**: 
- The super user feature uses Google OAuth for authentication
- You must enable Google Sign-In in your Firebase Console
- Only the specified email address will have super user access
- Super users can only reset admin passwords - they cannot access regular admin features
